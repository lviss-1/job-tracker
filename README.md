# Job Tracker
A full-stack job application tracker built to develop hands-on backend experience end-to-end.

## Tech Stack
- **Backend:** Spring Boot (REST API)
- **Frontend:** Next.js

## Overview
This project tracks job applications, interview stages, and company details through a Spring Boot REST API connected to a Next.js frontend.
The primary goal is to learn how to design and build a backend from scratch and integrate it with a modern frontend.

## Goals
- Design and build a REST API with Spring Boot
- Manage data persistence with a relational database
- Connect a Next.js frontend to a live backend API
- Learn full-stack architecture hands-on

## Getting Started

### Prerequisites
- Java 17+
- Node.js 18+
- npm or yarn

### Backend
```bash
cd backend
./mvnw spring-boot:run
```
